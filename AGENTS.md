# Cybertechnika Site Agent Instructions

This project is a personal DevSecOps portfolio for cybertechnika.pro.

The visual identity is:
- Premium cyberpunk
- Monokai Pro inspired
- Serious but expressive
- Technical, tactical, stylish
- JDM drift-car energy
- Ghost in the Shell inspired atmosphere
- NOT childish
- NOT corporate bland
- NOT edgy hacker cringe

The site should feel like:
"the dashboard of an elite infrastructure engineer."

Core principles:
- Everything as Code
- Automate Everything
- Security First
- Zero Trust
- Observability Matters
- Infrastructure should be elegant and understandable

Preferred stack:
- Next.js
- Tailwind CSS
- TypeScript
- Framer Motion

Avoid:
- Matrix rain
- Fake hacking visuals
- RGB overload
- Meme aesthetics
- Dashboard clutter
- Over-animation
