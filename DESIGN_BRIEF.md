# Cybertechnika Portfolio Design Brief

## Brand Direction

Cybertechnika is a premium DevSecOps and infrastructure engineering portfolio with a strong cyberpunk and JDM-inspired visual identity.

The site should feel like:
- a tuned machine
- an elite infrastructure dashboard
- precise engineering with personality
- technical but stylish
- calm but powerful

## Visual Style

Inspired by:
- Monokai Pro
- Ghost in the Shell atmosphere
- Neo Tokyo
- Tactical UI systems
- JDM drift culture
- Modern observability dashboards

Avoid:
- generic startup SaaS
- gamer aesthetics
- cringe hacker themes
- excessive anime
- over-the-top effects

## Colour Palette

Primary background:
- #0D1117
- #101418
- #151A21

Accent colours:
- Cyan: #66D9EF
- Green: #A6E22E
- Orange: #FD971F
- Pink: #F92672

## Site Structure

### Hero Section
- cinematic visual impact
- terminal-style intro
- strong identity statement
- subtle animated topology graphics

### Engineering Philosophy
- Everything as Code
- Security First
- Automation
- Observability

### Capability Grid
- Infrastructure
- Cloud
- Security
- Automation
- Identity
- DevOps

### Projects
Infrastructure-focused projects with:
- architecture visuals
- operational outcomes
- security improvements
- automation details

### Technical Stack
Displayed partially as:
- YAML
- config files
- deployment manifests

### Architecture Section
Explain:
- how the site is hosted
- CI/CD approach
- security posture
- deployment model

## Overall Goal

The site should make visitors think:

"This person clearly knows infrastructure engineering deeply, but also has personality and style."
